display-box

The display-box property mentions whether the current element and it's children generate a box.

none

This option makes it so that neither the element or any of it's children show up in the box tree at all.

Example(?):
An item
A child

contents

This option makes it so that the element does not show up in the box tree.

Example(?):
Some text before
A child
Some text after
