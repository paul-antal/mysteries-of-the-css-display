display-inside

The display-inside value dictates the layout of the direct child elements.

flow

The flow layout represents the default layout. It places inline elements in the writing direction of the used language, and block elements perpendicular to it.

table

The table layout is a composite layout that works exactly as the table elements do. For it to work it needs multiple other html elements with a specific display type to correspond to the different table elements.

Here is a list of table display types and their corresponding element:

  • table-row - tr
  • table-row-group - tbody
  • table-header-group - thead
  • table-footer-group - tfoot
  • table-column - col
  • table-column-group - colgroup
  • table-cell - td
  • table-caption - caption
