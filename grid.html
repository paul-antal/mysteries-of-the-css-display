And many more. Check them out in this guide Short cheat list:

justify-items - align elements horizontally in their cell : start, end, center, stretch

align-items - align elements vertically in their cell : start, end, center, stretch

justify-content - align the grid horizontally in it's parent : start, end, center, stretch, space-around, space-between, space-evenly

align-content - align the grid vertically in it's parent : start, end, center, stretch, space-around, space-between, space-evenly

grid-auto-columns / grid-auto-rows specify the size of any auto generated columns / rows

grid-auto-flow defines how elements will be automatically placed in the grid (flex-direction) : row / column dense (fill in previous empty spaces)
